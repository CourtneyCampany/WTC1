
All leaf count data, **including the April 2008 data**, which is also in the database in 
the directory "./Biomass and tree data/Architecture April 2008".

For the April 2008 set, all leaves on all branches were counted, on subsequent sets by Burhan et al.,
branches were sampled throughout the crowns.
